# Data Analysis Dashboard

To explore the thematic data from the **OneHealth EU** project, please use the interactive dashboard below.

---

### [🚀 Launch Interactive Dashboard](https://eu-health-ed-repository.streamlit.app/)
*Note: Replace with your actual Streamlit Cloud URL after deployment.*

---

### Dashboard Features:
- **Concept Explorer**: Filter by One Health (OH) or Planetary Health (PH).
- **Thematic Trends**: Analyze results by topic (Nutrition, Climate, AMR).
- **Transversal Mapping**: Review specific raw text snippets from across the EU.

*The dashboard is powered by Streamlit and visualizes the `analysis.jsonl` data stored in each country's folder.*
