## Instructions for Experts

1.  **Select your country** from the table above.
2.  **Review Section 1**: Validate the document list against the inclusion/exclusion criteria.
3.  **Review Section 2**: Verify the AI-extracted thematic snippets and translations.
4.  **Provide Feedback**: Use the qualitative feedback section (Section 3) to add bioregional insights or correct terminology.
5.  **Sign-off**: Mark the verification checkbox once the data is confirmed.

> [!TIP]
> If your country is not yet listed, it means the initial data extraction is still in progress. The repository will eventually cover all 27 EU Member States.
